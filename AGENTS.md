# syndicus

## Commands
- `npm run dev` - Start dev server on port 3000
- `npm run build` - Build for production

## Stack
- React 19 + TanStack Start (SSR)
- TanStack Router (file-based routing)
- Vite 8

## Key files
- `src/router.tsx` - Router factory
- `src/routeTree.gen.ts` - Auto-generated route tree (do NOT edit)
- `src/routes/` - Route components

## Generated code
routeTree.gen.ts is auto-generated by TanStack Router. Re-run codegen if routes change:
```bash
npx @tanstack/react-router generate
```

## Note
No test/lint/typecheck scripts in package.json.
